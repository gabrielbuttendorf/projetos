To Do List

Este é meu primeiro projeto desenvolvido com React. Trata-se de uma aplicação simples para gerenciar tarefas, onde é possível adicionar, marcar como concluídas, remover e pesquisar tarefas. O app também armazena as tarefas no localStorage para manter os dados mesmo após fechar o navegador. Além disso, conta com filtros para visualizar todas as tarefas, apenas as concluídas ou as pendentes, e possui uma interface responsiva, adaptada para dispositivos móveis.
